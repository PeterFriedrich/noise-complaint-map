## 4. Technical Learnings

- Dataset `ypje-j649` is "Bylaw Complaint Details" ‚Äî covers ALL bylaw complaint types. Must always filter `complaint_category = 'Noise'` in every pipeline query or you'll get the full dataset.
- Socrata field names in `docs/DATA.md` are inferred from display names, not confirmed by a live query. Verify with a small test query (`$limit=5`) before building the pipeline against them.
- The dataset URL for queries is `https://data.edmonton.ca/resource/ypje-j649.json` (not the `/api/v3/views/` URL the user originally shared ‚Äî that returns the full unfiltered dataset and exceeds fetch limits).
- `.env` holds `SOCRATA_API_KEY_ID` and `SOCRATA_API_KEY_SECRET`. Keys were rotated once ‚Äî old keys shared in chat, new keys stored in `.env`. Do not use the old keys.
- Oracle Ampere is ARM (aarch64) ‚Äî any Python dependencies with C extensions must have ARM-compatible wheels.

## 4. Technical Learnings

**deck.gl rendering issue (significant):**
- `deck.MapboxOverlay` with `interleaved: false` (default) does not render in Firefox — canvas is created but nothing draws. Tried `interleaved: true`, also didn't render.
- Fix: use `deck.Deck` standalone with a manually created canvas appended to `#map`. Pass canvas element via `canvas:` parameter (NOT `container:` — that's not a valid Deck param). Canvas must have `position:absolute;top:0;left:0;width:100%;height:100%` and `onResize` to set canvas dimensions.
- Working code in `site/js/map.js:18-28`.

**Socrata auth:**
- Credentials in `.env` are `SOCRATA_API_KEY_ID` (key ID) and `SOCRATA_API_KEY_SECRET` (secret).
- Must use **HTTP basic auth** (`requests.get(url, auth=(key_id, key_secret))`). The `X-App-Token` header does NOT work with these credentials.

**Protomaps CDN issue:**
- `https://unpkg.com/protomaps-themes-base@4/dist/index.js` is served as `text/plain` by unpkg — Firefox blocks it with MIME type mismatch. This script is not needed until Phase 0 (basemap setup). Removed from `index.html`.

**GitHub Pages constraints:**
- GitHub Pages API only allows serving from `/` or `/docs` subfolder. Since `docs/` is already used for project documentation, switched to GitHub Actions deploy workflow that copies `site/` to Pages. Workflow at `.github/workflows/deploy.yml`.

**Data findings (see `docs/FINDINGS.md`):**
- 14,656 records, zero missing lat/lon
- Weekdays have MORE complaints than weekends — Tuesday/Wednesday peak, Saturday lowest (counterintuitive)
- Dataset updates daily (confirmed via Socrata API metadata)
- `type_of_complaint` is always `"Noise"` — no useful subcategory
- `count` is always `"1.0"` — one row per complaint, not pre-aggregated
- 13.3% missing `neighbourhood` — doesn't matter since we bin by lat/lon

**MapLibre basemap (deferred — Phase 0):**
- Plan: self-host `edmonton.pmtiles` on Cloudflare R2 (free tier, range-request-safe)
- GitHub Pages has documented flaky range-request support for `.pmtiles` files
- Once R2 is set up, add MapLibre GL + pmtiles.js back to `index.html`, wire `BASEMAP_URL` in `site/js/config.js`
- `buildStyle()` was removed from `map.js` this session — needs to be re-added when basemap is ready
